Design Addendum

This section explains the reasoning behind the stylistic choices made in the development of this webpage. These choices were made with the goal of improving readability, creating a modern look, and enhancing user experience.

Font Choices

The font used throughout the page is Arial, which is a clean and widely used sans-serif font. It ensures the text is legible on all screen sizes and devices, providing a simple yet professional look. Arial is a standard web-safe font, making it a reliable choice for consistency across browsers.

Heading and Main Text Sizes

The main heading (h1) is set to a large font size of 2.5rem to make it stand out and serve as the focal point of the page. The letter-spacing of 2px ensures that the title has adequate space between each character, making it easier to read. The other headings (h2, h3) have progressively smaller sizes to create a clear content hierarchy.

The main body text is set to 1rem, a comfortable size for reading on all devices, providing a balance between legibility and aesthetics.

Color Choices

The color scheme was chosen to create a clean and modern aesthetic with high contrast for readability. The header and footer use a dark background color (#333), which contrasts with white text for easy readability. The navigation menu uses a slightly lighter shade of gray (#444) to differentiate it from the header while maintaining a cohesive look.

The hover effect on the navigation links uses a bright yellow color (#f4b400) to make it interactive and noticeable, enhancing the user experience.

The body background is set to #f4f4f4, a light gray color that provides a subtle background without being too bright or harsh on the eyes.

Margin and Padding Choices

The margin and padding are carefully adjusted to ensure sufficient space between elements, giving the page a balanced, clean, and spacious feel. The body has a default padding around all elements for proper spacing. The header and footer have padding of 1rem for balance and easy readability.

For sections, padding of 2rem is added to create enough space inside the content areas, while a margin of 20px is used between sections to prevent them from appearing cramped.

The navigation links have 10px of padding on each side and a 15px margin between them to create adequate clickable space and prevent crowding. The hover effect is enhanced by these padding and margin choices to ensure smooth interaction.
